これはRDTOOL外販用のプラットフォームとして開発予定

ディレクトリ構成は以下を想定

http://xxx.com / rd.php
               / media /      広告掲載メディアが実績を確認するための画面群
               / manager /　　広告主（サイトオーナー）もしくは広告代理店が実績を確認する画面
               / admin /　　　システム管理者が情報を確認する画面（例：SQLコマンドを入力・トラフィックやプロセスの監視・ユーザー追加など）


データベースは、　MySQL（MariaDBに対する互換性をもたせる）
DBへの接続方法は　PDOを使う。
SQLインジェクション対策として、プリペアドステートメントを使う。
セッションIDを使って　クリックユーザーごとの広告効果を測定する
管理画面の認証管理は、ユーザー名とメディア名等の組み合わせを暗号化し、Cookieを使って管理する
